자바 관련 프로그램을 작성하다 보면, 개발자의 실수로 코드 작성 예외가 발생한다.
발생하는 예외들 중 RuntimeException를 상속 받은 예외들이 있는데,
이런 런타임 예외를 상속 받은 예외 3가지 이상 적고,
발생하는 이유를 함께 서술 하세요.

1.예외: IndexOutOfBoundsException
1.이유: 배열에서 정해진 인덱스를 초과하여 접근하려는 경우에 발생하는 예외입니다. 예를 들면 -1 인덱스에 접근하거나
		총 길이가 10인 배열 즉 0 ~ 9까지 가지고 있는 배열에서 10에 접근하거나 하는 경우에 발생하는 예외입니다

2.예외: NullPointerException
2.이유: 할당 받지 못한 객체에 대해서 접근하려는 경우에 발생하는 예외입니다.
		String tmp; => 이와 같이 문자열 객체명만 선언해놓고 해당 tmp변수로 문자열 객체에 접근하려고 할
		경우에 발생하는 예외입니다.

3.예외: ArithmeticException
3.이유: 산술적 오류가 발생하는 경우에 발생하는 예외입니다. 가장 대표적인 예로는 어떠한 수를 0 으로 나누려고 할
		경우에 발생하는 예외입니다.
