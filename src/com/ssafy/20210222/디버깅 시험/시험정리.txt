<Comparable과 Comparator>
- 공통정보: 오름차순이면 해당 객체를 앞에다 넣고, 내림차순이면 뒤에다 넣음

- Comparator: 익명중첩클래스
1. 점수를 오름차순
Arrays.sort(s, new Comparator<Student>() {
	@Override
	public int compare(Student o1, Student o2) {
		return o2.score - o1.score;
	}
}
2. 이름으로 오름차순이면 compare 내부모습
o1.name.compareTo(o2.name);
3. 학점으로 내림차순, 학점이 같으면 학번으로 오름차순이면 compare 내부모습
if (o1.score==o2.score)
	return o1.num - o2.num;

- Comparable: 해당 객체가서 implements Comparable<객체클래스이름>
1. int형 배열일 경우 바로 Arrays.sort(arr) 가능
2. 객체형 배열일 경우 ClassCastException 나기 때문에 Comparable을 꼭 implement
그리고 Student 객체내에서 꼭 정의
public int compareTo(Student o)
	return Integer.compare(o.num, this.num);
